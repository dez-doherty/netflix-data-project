## Description
Exploratory data analysis on a Kaggle dataset of 8,800+ Netflix titles, covering data cleaning, processing and visualisations to uncover content trends across movies and TV shows.

## Technologies
- `Kaggle Datasets`
- `MSSQL`
- `Excel (Power Query + Data Model)`
- `Power BI`
  
## Features
- `Cleans raw CSV data and handles missing directors, cast, country and date fields`
- `Transforms the flat CSV into a relational data model using Power Query (normalising genres, countries and cast into separate dimension tables)`
- `Breaks down content split between Movies and TV Shows across release years`
- `Analyses rating distributions (TV-MA, TV-14, PG-13 etc.) by content type`
- `Visualises top producing countries and how Netflix's library has grown over time`
- `Explores genre trends across the catalogue`
